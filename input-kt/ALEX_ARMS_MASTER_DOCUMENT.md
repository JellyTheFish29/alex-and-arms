# 6. Keyword Mapping Framework

## Google Sheets Structure

### Tab 1 -- Pillar Pages

Columns: Page Name \| URL \| Primary Keyword \| Secondary Keywords \|
Volume \| Difficulty \| Intent \| Priority \| Status

### Tab 2 -- Category Mapping

Parent \| Category \| URL \| Primary Keyword \| Volume \| Difficulty \|
Standalone? \| Priority

### Tab 3 -- Brand Mapping

Brand \| URL \| Primary Keyword \| Volume \| Demand? \| Priority

### Tab 4 -- Raw Keyword Research

Keyword \| Volume \| Competition \| CPC \| Intent \| Cluster

### Tab 5 -- Architecture Decision Log

Date \| Page \| Final URL \| Target Keyword \| Reason \| Redirect Impact

## Core Rule

One page = one primary keyword cluster.

Commercial intent prioritized.
